# Retrovue

This repository contains the source code for Retrovue, a multi-component application for video playout.

## Overview

This project is organized as a monorepo and contains the following main components:

-   `pkg/core`: The Python-based coordinator and main application logic.
-   `pkg/air`: The C++ based real-time playback engine.
-   `docs/`: Project documentation and standards.
